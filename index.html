The hardest part already works

Demoed at a Stellar Developer Meeting, on mainnet

Scan a Vietnamese QR → quote via Gaian's API → real USDC settles on Stellar mainnet → fiat pays out through a regulated PSP. On-chain, verifiable, in production.

What this grant funds

That rail is a closed, custom integration — no third-party Stellar wallet can use it, because it doesn't yet speak Stellar's standard protocols (the SEPs).

Standardizing it into a public anchor is the net-new work — a working system to standardize, not a build from zero.

Result: the largest phase carries the lowest delivery risk in the grant.
